# Overview
Custom web rhythm game built through Unity. Travel through different countries 
and explore their music and culture through increasingly difficult custom-made levels. 
- Can support up to 4 players.
- 100% custom art and music.

## Usage
Access the following link: vishalsund.github.io/rhythm-quest/
- The game will have a beginning cutscene.
- Access the tutorial after the cutscene by clicking on the bottom
right compass on the main menu.
- Make sure to click on at least one player on the selection screen
before starting, or the level won't show properly.

## Requirements
- Keyboard
- Recommended: Sound Output
